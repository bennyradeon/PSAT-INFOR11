Cara Mengganti Warna

Cari bagian :root di dalam <style>, lalu ubah nilai warna sesuai tema website.

:root {
    --warna-utama: #0056b3;
    --warna-kedua: #ff5733;
    --warna-bg: #f4f8ff;
}

Cara Mengganti Gambar

Cari tag <img src="...">, lalu ganti URL gambar atau gunakan gambar lokal, misalnya gambar/foto.jpg.

Bagian yang Sebaiknya Tidak Dihapus

  • <nav> untuk menu navigasi.
  • <header> untuk bagian pembuka atau hero.
  • <main> dan <section> untuk isi utama.
  • <footer> untuk bagian penutup.
  • Media query @media agar tampilan tetap responsif di HP.

Template Laporan Project

Gunakan template laporan berikut untuk mendokumentasikan project. Kerangka laporan sudah disiapkan agar siswa tinggal menyesuaikan isi dengan website yang dibuat.

Cara menggunakan:

  1. Klik tombol Buka Template Laporan.
  2. Pilih menu File → Buat Salinan.
  3. Edit sesuai website yang dibuat, lalu ekspor sebagai PDF.
Buka Template Laporan

Unduh Template dari Google Drive

File HTML juga dapat diunduh melalui folder Google Drive berikut.

Buka Folder Google Drive

Cara download:

  1. Login dengan akun Google.
  2. Pilih file template yang ingin digunakan.
  3. Klik kanan, lalu pilih Unduh.
  4. Buka di VS Code, edit, lalu simpan sebagai index.html.
